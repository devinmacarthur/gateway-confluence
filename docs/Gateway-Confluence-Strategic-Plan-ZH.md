## 东 Portland 社区战略计划

**地点：** Gateway 区，Portland OR 97220
**中心地址：** 1708 NE 106th Avenue, Portland, Oregon
**状态：** 草案 — 社区审阅中

---

## 我们是谁

Gateway Confluence 是东 Portland Gateway 区的一项社区驱动计划。我们正在建设一个以公共交通为中心、文化多元、经济稳定的社区——让长期居民能够留下来，小企业能够在不被迫迁的情况下繁荣发展，公共空间促进归属感，基础设施服务于日常生活而非投机性增长。

这不是一个旧城改造计划。这是一个让人们留下来的计划。为已经生活在这里的人而制定。

---

## 我们的社区

Gateway 是 Oregon 州最多元化的社区之一：

- 使用**超过40种语言**
- 居民来自**超过80个国家**
- 拥有**超过50,000名**居民
- 拥有**超过500家**小企业，其中60%为移民所有
- **7大战略支柱**引领我们的工作

Gateway 多语言、多文化的特质是我们最大的优势。我们放大这种力量——而不是去管理它。

---

## 我们的价值观

**反迫迁** — 发展不得迫使现有居民或企业搬离。每个项目都应巩固已有的社区。

**文化多元** — Gateway 的多样性不是需要解决的问题，而是需要在其上建设的基础。

**社区治理** — 居民自己决定社区的事务。不是咨询建议，而是真正的权力。

**基础设施即尊严** — 托儿服务、食物获取、降温中心和职业培训是权利，不是福利设施。

**以公共交通为中心的公平** — 公共交通应当连接人们与机会，而不是将人们排挤出去。

## 七大战略支柱

### 1. 住房保障——杜绝迫迁
*优先级：紧迫*

Gateway 52%的租户将超过30%的收入用于支付房租，25%的租户则将超过一半的收入用于房租。

**目标：**
- 现有居民零净迫迁
- 到2032年建立200个单元的社区土地信托
- 实施租户优先购买权政策
- 保留1,000个以上现有可负担住房单元

**关键行动：**
- 建立 Gateway Community Land Trust (GCLT)
- 租户保护和回归权项目
- 现有建筑保护——3年内覆盖500个单元
- 为家庭创建住房稳定通道

---

### 2. 小企业生态系统
*优先级：高*

Gateway 的小企业——餐车、维修店、美容院、杂货店——是该区域经济和文化的支柱。大多数企业的利润率仅为3-5%。

**目标：**
- 50家企业获得稳定的长期租约
- 提供5种以上语言的多语言商业支持
- 3年内发放50万美元小额补助金
- 启动3个工人所有的合作社

**关键行动：**
- 多语言商业支持中心
- Gateway 小额补助金和小额贷款项目
- 合作社发展孵化器
- 商业租约稳定化倡议

---

### 3. 公共空间激活
*优先级：高*

东 Portland 的人均公园面积比全市其他地区少40%。Gateway 是为汽车建造的，而不是为人。

**目标：**
- 3个激活的公共广场
- 连接公共交通与日常生活的安全步行通道
- 在2个以上地点设立青年设计的空间
- 沿主要道路建设绿色缓冲带

**关键行动：**
- 重新构想 Gateway Transit Plaza
- 在10个最危险的十字路口设置安全人行横道
- 绿色通廊——第一阶段种植500棵树
- 面向12-24岁青年的青年主导空间设计

---

### 4. 社区安全
*优先级：高*

当居民谈论安全时，他们谈论的是黑暗的街道、危险的路口和孤立感——而不是犯罪统计数据。

**目标：**
- 全区域综合街道照明
- 20名经过培训的多语言社区大使
- 每周7天的创伤知情外展服务
- 行人伤亡减少50%

**关键行动：**
- Gateway 照明倡议
- 社区大使项目
- 创伤知情外展服务扩展
- "街道之眼"商业激活计划

---

### 5. 机构协调
*优先级：高*

超过15个公共机构和30多个非营利组织在 Gateway 运营——但只有3个之间有正式的协调协议。

**目标：**
- 由居民主导的治理委员会，拥有正式的咨询权力
- 所有机构合作伙伴采纳共同的社区愿景
- 协调投资框架
- 年度社区问责峰会

**关键行动：**
- 建立 Gateway Community Governance Council
- 各机构采纳共同的社区愿景
- 创建协调投资框架
- 举办年度问责峰会——居民提问，机构回答

---

### 6. 文化可见性
*优先级：高*

来自80个国家，使用40多种语言。但 Gateway 的文化丰富性在正式体系中却是隐形的。

**目标：**
- 全区域多语言导向标识
- 10件由本地艺术家创作的公共艺术装置
- 年度多元文化节——5,000人以上参与
- 社区文化遗产档案馆

**关键行动：**
- Gateway 文化艺术项目
- 多语言导向标识系统
- Gateway 遗产档案——收集口述历史
- Gateway Cultures Festival——庆祝每一个社区

---

### 7. 共享基础设施
*优先级：紧迫*

Gateway 在基础设施方面存在根本性的缺口：托儿服务、职业培训、食物获取和气候韧性。

**目标：**
- 为500多个家庭提供可负担的托儿服务
- 每年为1,000名居民提供职业培训
- 社区食物枢纽
- 4个降温和韧性中心
- 多语言心理健康服务

**关键行动：**
- Gateway 托儿合作网络
- Gateway 就业中心——多语言职业培训
- 社区食物枢纽——连接种植者、厨房和家庭
- 极端高温降温中心和韧性中心
- 多语言心理健康倡议

## 如何参与

**志愿服务** — 加入与七大战略支柱之一对应的工作小组。我们需要组织者、翻译人员、研究人员和社区联络人。

**参加会议** — 我们的月度社区会议对所有人开放。提供西班牙语、越南语、中文和俄语的口译服务，并提供儿童看护。

**分享您的故事** — 您的经历塑造着这份计划。无论您在 Gateway 生活了数十年还是刚刚搬来，您的看法都很重要。

**传播信息** — 告诉您的邻居关于 Gateway Confluence 的信息。分享我们的网站。邀请他人参加会议。社区的力量在一次次对话中不断壮大。

## 工作小组

每个工作小组对应一个战略支柱：

1. **住房保障** — 反迫迁、租户权利、社区土地信托
2. **小企业** — 小额补助金、合作社发展、租约稳定化
3. **公共空间** — 交通广场、安全人行横道、绿色通廊
4. **社区安全** — 照明、社区大使、创伤知情外展
5. **治理** — 社区治理委员会、机构问责
6. **文化可见性** — 公共艺术、多语言标识、遗产档案
7. **共享基础设施** — 托儿服务、就业培训、食物枢纽、降温中心
